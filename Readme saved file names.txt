**********************************************************
Files are saved in the following format: [Automation function prefix]_[Reading function]_[order no of automated measurement]_[current time].csv
E.g. Rar_Retention_1_632224124 in Log_Data folder
**********************************************************
For more reading functions applied in a row during an automation function, they will be issued with the same order number.
**********************************************************

Files saved with the prefixes mentioned below represent the automated function that collected a given set of readings:

Rar - data comes from a set of readings with Read any row function
Rrr- data comes from a set of readings with Read row range function
Rfw- data comes from a set of readings with Read full wafer function
Rdf - data comes from a set of readings with Read redined function
